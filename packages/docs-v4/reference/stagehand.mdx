icon: "hand-horns"
---

`Stagehand` owns the browser connection and exposes the session's top-level lifecycle and diagnostics.

<View title="TypeScript" icon="js">

## Quick start

```typescript
import { Stagehand } from "@browserbasehq/stagehand";

const stagehand = new Stagehand({ browser: { type: "local" } });
await stagehand.init();
```

## init()

Initialize Stagehand and connect to the configured browser.

```typescript
await stagehand.init();
```

<ResponseField name="result" type="Promise<void>">
  Resolves after the operation completes.
</ResponseField>

## close()

Close the Stagehand session and release browser resources.

```typescript
await stagehand.close();
```

<ResponseField name="result" type="Promise<void>">
  Resolves after the operation completes.
</ResponseField>

## ping()

Check that the Stagehand runtime is responding.

```typescript
const status = await stagehand.ping();
```

<ResponseField name="result" type="Promise<StagehandPingResult>">
  The operation result.

<ResponseField name="result.ok" type="boolean">
  Whether the runtime responded successfully.
</ResponseField>

  <ResponseField name="result.runtime" type="string">
    The responding runtime.
  </ResponseField>
</ResponseField>

## runtimeLoopbackStatus()

```typescript

```
