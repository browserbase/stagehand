A `Page` combines deterministic browser controls with Stagehand's natural-language `act`, `observe`, and `extract` methods.

<View title="TypeScript" icon="js">

## Quick start

```typescript
const page = await stagehand.context.newPage();
await page.goto("https://example.com");
```
