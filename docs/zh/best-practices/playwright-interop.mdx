Stagehand 构建于 [Playwright](https://playwright.dev/) 之上，因此你可以通过 Stagehand 实例直接调用 Playwright 的方法。

<div id="page-and-context">
  ## `page` 和 `context`
</div>

`stagehand.page` 和 `stagehand.context` 分别是 Playwright 的 `Page` 和 `BrowserContext` 实例。使用这些接口与 Stagehand 所使用的 Playwright 实例进行交互。

<CodeGroup>
```TypeScript TypeScript
const page = stagehand.page;
// Playwright 的基础方法可用
await page.goto("https://github.com/browserbase/stagehand");

// Stagehand 对 Playwright 对象进行了扩展/重写
await page.act("click on the contributors")
```

```python Python
page = stagehand.page
# Playwright 的基础方法可用
await page.goto("https://github.com/browserbase/stagehand")

# Stagehand 对 Playwright 对象进行了扩展/重写
await page.act("click on the contributors")
```
</CodeGroup>
