Stagehand 构建在 [Playwright](https://playwright.dev/) 之上，因此你可以通过 Stagehand 实例直接调用 Playwright 的方法。

## `page` 和 `context`

`stagehand.page` 和 `stagehand.context` 分别是 Playwright 的 `Page` 与 `BrowserContext` 实例。使用它们与 Stagehand 所依托的 Playwright 实例进行交互。

<CodeGroup>
```TypeScript TypeScript
const page = stagehand.page;
// 原生 Playwright 方法可用
await page.goto("https://github.com/browserbase/stagehand");

// Stagehand 对 Playwright 对象进行了扩展
await page.act("click on the contributors")
```

```python Python
page = stagehand.page
# 原生 Playwright 方法可用
await page.goto("https://github.com/browserbase/stagehand")

# Stagehand 对 Playwright 对象进行了扩展
await page.act("click on the contributors")
```
</CodeGroup>
