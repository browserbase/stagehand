## 概览

本指南将介绍如何将 Stagehand 与 LangChain JS 搭配使用，创建可自动执行网页交互的智能代理。读完后，你将能够：

- 使用 LangChain JS 配置 StagehandToolkit
- 创建可在网站中导航并进行交互的代理
- 通过自然语言指令提取结构化数据
- 使用 LangGraph 构建复杂的自动化工作流

## 适用场景

Langchain JS 集成非常适合需要具备高级推理能力的智能网页自动化：

- **AI 驱动的研究**：创建代理在多个网站上搜集信息并整合结论
- **动态表单填写**：根据上下文需求自动填写复杂表单
- **数据提取流程**：借助智能导航从多个来源提取并转换数据
- **多步网页流程**：执行需要决策与自适应的复杂浏览器流程
