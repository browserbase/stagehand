## 概览

本指南将介绍如何将 Stagehand 与 LangChain JS 结合使用，构建能够自动化网页交互的智能代理。阅读完本指南后，你将了解如何：

- 在 LangChain JS 中设置 StagehandToolkit
- 创建可导航并与网站交互的代理
- 使用自然语言指令提取结构化数据
- 借助 LangGraph 构建复杂的自动化工作流

## 适用场景

Langchain JS 集成非常适合需要高级推理能力的智能网页自动化：

- **AI 驱动的研究**：创建代理在多个网站上检索信息并汇总结论
- **动态表单填写**：根据上下文自动填写复杂表单
- **数据提取工作流**：通过智能导航从多个来源提取并转换数据
- **多步骤网页流程**：执行需要决策与自适应的复杂浏览器工作流
