## 概览

Browserbase MCP Server 通过模型上下文协议（MCP）为 Claude 提供强大的浏览器自动化能力。基于 [Stagehand](https://docs.stagehand.dev/) 构建，此集成通过自然语言指令实现 AI 驱动的网页自动化。

<Info>
  该服务器使 Claude 能够控制浏览器、导航网站、与网页元素交互并提取数据——且这一切都可通过简单的对话式指令完成。
</Info>

## 会话管理

<Info>
  Browserbase MCP Server 同时支持单会话与多会话架构，以满足不同的自动化需求。
</Info>

<Tabs>
  <Tab title="Single Session Mode">
    **传统方案**

    - 同一时间仅有一个活动的浏览器会话
    - 适用于基础自动化任务
    - 自动管理会话生命周期
    - 适合顺序式工作流程
  </Tab>

  <Tab title="Multi-Session Mode">
    **高级并行处理**

    - 多个彼此独立的浏览器会话
    - 每个会话各自维护独立状态
    - 支持并行执行
    - 非常适合复杂工作流程
  </Tab>
</Tabs>

## 入门

<Steps>
  <Step title="安装 MCP 服务器">
    根据你的需求，选择通过 NPM 安装、使用远程托管的 URL，或在本地进行开发。
  </Step>

  <Step title="配置身份验证">
    在 MCP 配置中设置你的 Browserbase API 凭据。
    在 [Browserbase 仪表板](https://www.browserbase.com/overview)获取你的 API 密钥。
  </Step>

  <Step title="开始自动化">
    开始使用自然语言指令，通过 Claude 控制浏览器。
  </Step>
</Steps>

<Tip>
  准备好开始了吗？查看我们的[设置指南](/zh/integrations/mcp/setup)，获取详细的安装说明。
</Tip>
