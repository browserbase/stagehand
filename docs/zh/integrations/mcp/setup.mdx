<Card title="Install with Cursor" icon="arrow-pointer" href="cursor://anysphere.cursor-deeplink/mcp/install?name=browserbase&config=eyJjb21tYW5kIjoibnB4IEBicm93c2VyYmFzZWhxL21jcCIsImVudiI6eyJCUk9XU0VSQkFTRV9BUElfS0VZIjoiIiwiQlJPV1NFUkJBU0VfUFJPSkVDVF9JRCI6IiIsIkdFTUlOSV9BUElfS0VZIjoiIn19">
在 Cursor 中一键安装，附带预配置
</Card>

我们支持为 MCP 服务器提供多种传输方式：STDIO 和 SHTTP。建议使用 SHTTP 并配合我们的远程托管 URL，以充分发挥服务器的性能。

## 前置条件

<Steps>
<Step title="获取你的 Browserbase 凭据">
在 [Browserbase 控制台](https://www.browserbase.com/overview)获取你的 Browserbase API 密钥和项目 ID。

<Frame>
<img src="/images/quickstart/api-key.png" alt="Browserbase API 密钥和项目 ID 设置" />
</Frame>

然后直接从输入框复制你的 API 密钥和项目 ID。
</Step>
</Steps>

## 安装方法

<Tabs>
<Tab title="远程 URL（SHTTP）">

前往 [smithery.ai](https://smithery.ai/server/@browserbasehq/mcp-browserbase)，输入你的 API 密钥和相关配置，即可获取远程托管的 URL。

![Smithery](../../images/mcp/smithery.jpg)

<CodeGroup>
```json Smithery
{
  "mcpServers": {
    "browserbase": {
      "url": "your-smithery-url.com"
    }
  }
}
```
</CodeGroup>

使用我们的远程托管服务器时，我们会承担 Gemini 的 LLM 成本；该模型是 [Stagehand](https://www.stagehand.dev) 中的[表现最佳模型](https://www.stagehand.dev/evals)。

</Tab>

<Tab title="NPM 包（STDIO）">
在本地入门的最简方式是使用我们的 NPM 包。

<Note>
如需使用其他模型，请在 args 中传入模型名称和密钥。更多信息见[此处](https://docs.browserbase.com/integrations/mcp/configuration)。
</Note>

<Steps>
<Step title="添加到 MCP 配置">
打开你的 MCP 配置 JSON，添加 Browserbase Server：

<CodeGroup>
```json Claude Desktop
{
  "mcpServers": {
    "browserbase": {
      "command": "npx",
      "args": ["@browserbasehq/mcp-server-browserbase"],
      "env": {
        "BROWSERBASE_API_KEY": "your_api_key",
        "BROWSERBASE_PROJECT_ID": "your_project_id",
        "GEMINI_API_KEY": "your_gemini_api_key"
      }
    }
  }
}
```
</CodeGroup>
</Step>

<Step title="重启 MCP 客户端">
<Check>
完成！重新加载 MCP 客户端后，Claude 即可使用 Browserbase。
</Check>
</Step>
</Steps>
</Tab>

<Tab title="本地开发">
用于本地开发或定制化时，你可以在本地运行服务器。

<Steps>
<Step title="克隆并构建">
```bash
# Clone the Repo
git clone https://github.com/browserbase/mcp-server-browserbase.git
cd mcp-server-browserbase

# Install the dependencies and build the project
npm install && npm run build
```
</Step>

<Step title="选择传输方式">
你可以通过 STDIO 或 Streamable HTTP（SHTTP）在本地运行。

<Tabs>
<Tab title="STDIO">
将以下内容添加到你的 MCP 配置 JSON 文件中：

```json
{
  "mcpServers": {
    "browserbase": {
      "command": "node",
      "args": ["/path/to/mcp-server-browserbase/cli.js"],
      "env": {
        "BROWSERBASE_API_KEY": "your_api_key",
        "BROWSERBASE_PROJECT_ID": "your_project_id",
        "GEMINI_API_KEY": "your_gemini_api_key"
      }
    }
  }
}
```
</Tab>

<Tab title="SHTTP（自托管）">
首先运行服务器：

```bash
node cli.js --port 8931
```

然后将以下内容添加到你的 MCP 配置 JSON 文件中：

```json
{
  "mcpServers": {
    "browserbase": {
      "url": "http://localhost:8931/mcp",
      "env": {
        "BROWSERBASE_API_KEY": "your_api_key",
        "BROWSERBASE_PROJECT_ID": "your_project_id",
        "GEMINI_API_KEY": "your_gemini_api_key"
      }
    }
  }
}
```
</Tab>
</Tabs>
</Step>

<Step title="重启客户端">
<Check>
重新加载 MCP 客户端即可开始使用。
</Check>
</Step>
</Steps>
</Tab>
</Tabs>

## 验证安装

<Steps>
<Step title="重启你的 Claude Client">
重启或刷新你的 Claude Client 应用，然后点击 🔨 图标即可看到可用工具。
</Step>

<Step title="测试集成">
让 Claude 导航至任意页面以开始使用我们的 MCP 服务器，并在[仪表盘](https://www.browserbase.com/sessions)查看你的 Browserbase 浏览器的实际运行情况。

<Tip>
可以这样对 Claude 说："Navigate to google.com and take a screenshot"
</Tip>
</Step>
</Steps>
