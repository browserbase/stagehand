我们为 MCP 服务器提供多种传输方式：STDIO 和 SHTTP。建议使用带有我们远程托管 URL 的 SHTTP，以充分发挥服务器性能。

## 先决条件

<Steps>
<Step title="获取你的 Browserbase 凭证">
从 [Browserbase 仪表板](https://www.browserbase.com/overview) 获取你的 Browserbase API 密钥和项目 ID。

<Frame>
<img src="/images/quickstart/api-key.png" alt="Browserbase API 密钥和项目 ID 设置" />
</Frame>

然后直接从输入框复制你的 API 密钥和项目 ID。
</Step>
</Steps>

## 安装方法

<Tabs>
<Tab title="远程 URL（SHTTP）">

前往 [smithery.ai](https://smithery.ai/server/@browserbasehq/mcp-browserbase)，输入你的 API 密钥和配置，以获取一个远程托管的 URL。

![Smithery](../../images/mcp/smithery.jpg)

<CodeGroup>
```json Smithery
{
  "mcpServers": {
    "browserbase": {
      "url": "your-smithery-url.com"
    }
  }
}
```
</CodeGroup>

使用我们的远程托管服务器时，我们会承担 Gemini 的 LLM 费用；在 [Stagehand](https://www.stagehand.dev) 中，它是[表现最佳的模型](https://www.stagehand.dev/evals)。

</Tab>

<Tab title="NPM 包（STDIO）">
本地入门的最简单方式是使用我们的 NPM 包。

<Note>
如果你想使用其他模型，需要在 args 中传入模型名称和对应密钥。更多信息见[此处](https://docs.browserbase.com/integrations/mcp/configuration)。
</Note>

<Steps>
<Step title="添加到 MCP 配置">
打开你的 MCP 配置 JSON，添加 Browserbase 服务器：

<CodeGroup>
```json Claude Desktop
{
  "mcpServers": {
    "browserbase": {
      "command": "npx",
      "args": ["@browserbasehq/mcp-server-browserbase"],
      "env": {
        "BROWSERBASE_API_KEY": "your_api_key",
        "BROWSERBASE_PROJECT_ID": "your_project_id",
        "GEMINI_API_KEY": "your_gemini_api_key"
      }
    }
  }
}
```
</CodeGroup>
</Step>

<Step title="重启你的 MCP 客户端">
<Check>
就这些！重新加载你的 MCP 客户端，Claude 就可以使用 Browserbase 了。
</Check>
</Step>
</Steps>
</Tab>

<Tab title="本地开发">
用于本地开发或自定义时，你可以在本地运行服务器。

<Steps>
<Step title="克隆并构建">
```bash
# 克隆仓库
git clone https://github.com/browserbase/mcp-server-browserbase.git
cd mcp-server-browserbase

# 安装依赖并构建项目
npm install && npm run build
```
</Step>

<Step title="选择传输方式">
你可以使用 STDIO 或 Streamable HTTP（SHTTP）在本地运行。

<Tabs>
<Tab title="STDIO">
将以下内容添加到你的 MCP 配置 JSON 文件中：

```json
{
  "mcpServers": {
    "browserbase": {
      "command": "node",
      "args": ["/path/to/mcp-server-browserbase/cli.js"],
      "env": {
        "BROWSERBASE_API_KEY": "your_api_key",
        "BROWSERBASE_PROJECT_ID": "your_project_id",
        "GEMINI_API_KEY": "your_gemini_api_key"
      }
    }
  }
}
```
</Tab>

<Tab title="SHTTP（自托管）">
首先，运行服务器：

```bash
node cli.js --port 8931
```

然后将以下内容添加到你的 MCP 配置 JSON 文件中：

```json
{
  "mcpServers": {
    "browserbase": {
      "url": "http://localhost:8931/mcp",
      "env": {
        "BROWSERBASE_API_KEY": "your_api_key",
        "BROWSERBASE_PROJECT_ID": "your_project_id",
        "GEMINI_API_KEY": "your_gemini_api_key"
      }
    }
  }
}
```
</Tab>
</Tabs>
</Step>

<Step title="重启你的客户端">
<Check>
重新加载你的 MCP 客户端，就可以开始使用了！
</Check>
</Step>
</Steps>
</Tab>
</Tabs>

## 验证安装

<Steps>
<Step title="重启你的 Claude 客户端">
重启或刷新你的 Claude 客户端应用，然后点击 🔨 图标即可看到可用工具。
</Step>

<Step title="测试集成">
让 Claude 导航到任意页面来开始使用我们的 MCP 服务器，并在[仪表板](https://www.browserbase.com/sessions)上查看你的 Browserbase 浏览器实际运行情况。

<Tip>
可以这样问 Claude："导航到 google.com 并截一张屏"
</Tip>
</Step>
</Steps>
