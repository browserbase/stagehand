## 系统要求

- 本地安装 **Node 18+**
- **模型密钥**：OpenAI 或 Anthropic（或接入自定义客户端）
- **Browserbase 密钥**：云端浏览器需提供 `BROWSERBASE_API_KEY` 和 `BROWSERBASE_PROJECT_ID`

<Tip>
  Vercel 上无法使用本地 Playwright 浏览器。部署时请将 Stagehand 设置为 Browserbase。
</Tip>
