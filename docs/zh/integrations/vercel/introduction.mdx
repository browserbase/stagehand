## 要求

- 本地安装 **Node 18+**
- **模型密钥**：OpenAI 或 Anthropic（也可接入自定义客户端）
- **Browserbase 密钥**：用于云浏览器的 `BROWSERBASE_API_KEY` 与 `BROWSERBASE_PROJECT_ID`

<Tip>
  Vercel 上无法使用本地 Playwright 浏览器。部署时请将 Stagehand 配置为使用 Browserbase。
</Tip>
