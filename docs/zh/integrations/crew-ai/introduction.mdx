## 概览

本指南介绍如何将 CrewAI 与 Browserbase 配合使用，构建可自动化网页交互的智能代理。完成本指南后，您将能够：

- 使用 StagehandTool 配置 CrewAI
- 创建可与网站交互的代理
- 通过自然语言指令自动化浏览器任务
- 从网页中提取结构化数据

## 适用场景

CrewAI 集成非常适合需要智能网页自动化的场景：

- **研究自动化**：让代理跨多个网站调研和收集信息
- **数据采集**：从电商、招聘或新闻网站提取结构化数据
- **表单自动化**：根据特定条件自动填写并提交表单
- **多步骤工作流**：执行需要决策的复杂浏览器流程

StagehandTool 封装了 Stagehand 的 Python SDK，使 CrewAI 智能代理能够控制真实浏览器，并通过三个核心原语与网站交互：

1. **Act**：执行点击、输入、导航等操作
2. **Extract**：从网页提取结构化数据
3. **Observe**：识别并分析页面元素
