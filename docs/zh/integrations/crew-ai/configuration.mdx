本指南将带你在 Browserbase 上完成 CrewAI 的设置，创建能够依据自然语言指令执行网页自动化任务的代理。

## 步骤 1：安装依赖

安装用于集成 CrewAI 与 Stagehand（AI 增强的浏览器自动化框架）的必要包：

```bash
pip install stagehand-py crewai crewai-tools
```

## 步骤 2：配置环境变量

你需要从以下服务获取 API 密钥：

1. **Browserbase API Key 和 Project ID**：在你的 [Browserbase 仪表盘](https://www.browserbase.com/) 获取
2. **LLM API Key**：从 [OpenAI](https://platform.openai.com/api-keys) 或 [Anthropic](https://console.anthropic.com/) 获取

将这些 API 密钥安全地保存为环境变量：

```bash
BROWSERBASE_API_KEY="your-browserbase-api-key"
BROWSERBASE_PROJECT_ID="your-browserbase-project-id"
OPENAI_API_KEY="your-openai-api-key"
ANTHROPIC_API_KEY="your-anthropic-api-key"
```

## 步骤 3：创建你的第一个 agent()

使用一个基础的 CrewAI agent()（自主多步骤浏览器代理）创建一个 Python 脚本：

```python
import os
from crewai import Agent, Task, Crew
from crewai_tools import StagehandTool
from stagehand.schemas import AvailableModel

# 从环境变量获取 API 密钥
browserbase_api_key = os.environ.get("BROWSERBASE_API_KEY")
browserbase_project_id = os.environ.get("BROWSERBASE_PROJECT_ID")
model_api_key = os.environ.get("OPENAI_API_KEY")  # 或 ANTHROPIC_API_KEY

# 初始化 StagehandTool
stagehand_tool = StagehandTool(
    api_key=browserbase_api_key,
    project_id=browserbase_project_id,
    model_api_key=model_api_key,
    model_name=AvailableModel.GPT_4O,  # 或 AvailableModel.CLAUDE_3_7_SONNET_LATEST
)

# 使用该工具创建一个 agent
researcher = Agent(
    role="Web Researcher",
    goal="查找并总结网站上的信息",
    backstory="我擅长在网上高效搜集信息。",
    verbose=True,
    tools=[stagehand_tool],
)
```

## 第 4 步：创建并运行任务

为你的 agent()（自主多步骤浏览器代理）定义一个任务并执行：

```python
# 创建一个使用该工具的任务
research_task = Task(
    description="访问 https://www.example.com，并告诉我首页的内容。",
    agent=researcher,
)

# 运行 crew
crew = Crew(
    agents=[researcher],
    tasks=[research_task],
    verbose=True,
)

try:
    result = crew.kickoff()
    print(result)
finally:
    # 释放资源
    stagehand_tool.close()
```

## 第 5 步：运行脚本

运行你的 Python 脚本：

```bash
python your_crew_script.py
```

## 高级配置

通过附加参数自定义 StagehandTool 的行为：

```python
stagehand_tool = StagehandTool(
    api_key=browserbase_api_key,
    project_id=browserbase_project_id, 
    model_api_key=model_api_key,
    model_name=AvailableModel.CLAUDE_3_7_SONNET_LATEST,
    dom_settle_timeout_ms=5000,  # 更长时间等待 DOM 稳定
    headless=True,  # 以无头模式运行浏览器
    self_heal=True,  # 尝试从错误中恢复
    wait_for_captcha_solves=True,  # 等待 CAPTCHA 解决
    verbose=1,  # 控制日志详细度（0-3）
)
```
