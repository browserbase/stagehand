Stagehand 是一款浏览器自动化框架，支持通过自然语言与代码来控制浏览器。它将 AI 的强大能力与代码的精确性结合起来，使网页自动化更灵活、易维护，且真正可靠。

## 浏览器自动化的痛点

像 Playwright 和 Puppeteer 这样的传统框架迫使你编写易碎的脚本，UI 每次改动都可能导致脚本失效。Web 代理宣称用 AI 解决这一问题，却让你不得不面对不可控的行为。

**你被困在两个糟糕的选项之间：**

- **过于脆弱**：传统选择器在网站变化时就会失效
- **过于“智能”**：AI 代理行为不可预测，且几乎无法调试

<CodeGroup>
  ```typescript TypeScript
  // Act - Execute natural language actions
  await page.act("click the login button");

  // Extract - Pull structured data
  const { price } = await page.extract({
    schema: z.object({ price: z.number() })
  });

  // Observe - Discover available actions
  const actions = await page.observe("find submit buttons");

  // Agent - Automate entire workflows
  const agent = stagehand.agent({
      provider: "anthropic",
      model: "claude-sonnet-4-20250514",
      options: {
        apiKey: process.env.ANTHROPIC_API_KEY,
      },
  })
  await agent.execute("apply for this job");
  ```

  ```python Python
  # Act - Execute natural language actions
  await page.act("click the login button")

  # Extract - Pull structured data
  result = await page.extract(
    schema={"price": float}
  )

  # Observe - Discover available actions
  actions = await page.observe("find submit buttons")

  # Agent - Automate entire workflows
  await agent.execute("apply for this job")
  ```
</CodeGroup>

## 开发者为何选择 Stagehand

- **精细掌控**：将 AI 驱动的操作与确定性代码灵活组合。你来精准拿捏 AI 的参与度。

- **真正可复现**：操作可原样保存并重放。告别浏览器自动化里“只在我机器上能跑”的尴尬。

- **可规模维护**：一套脚本可自动化多个网站。站点变动时，你的自动化会相应适配。

- **可组合式工具**：借助 Act、Extract、Observe 和 Agent，自由选择所需的自动化层级。
