Stagehand 是一款浏览器自动化框架，可通过自然语言与代码控制网页浏览器。通过将 AI 的能力与代码的精确性相结合，Stagehand 让网页自动化更加灵活、易维护且真正可靠。

## 浏览器自动化的问题

像 Playwright 和 Puppeteer 这样的传统框架会迫使你编写脆弱的脚本，任何 UI 变化都可能导致脚本失效。Web 代理声称用 AI 解决这一问题，但会让你陷入不可预测的行为之中。

**你被困在两个糟糕的选项之间：**

- **过于脆弱**：网站变化时，传统选择器会失效
- **过于“代理”**：AI 代理行为不可预测，且几乎无法调试

## 认识 Stagehand

Stagehand 通过四个强大的原语，让你在“二者兼得”的同时，精确掌控所用 AI 的程度：

<CodeGroup>
  ```typescript TypeScript
  // Act - Execute natural language actions
  await page.act("click the login button");

  // Extract - Pull structured data
  const { price } = await page.extract({
    schema: z.object({ price: z.number() })
  });

  // Observe - Discover available actions
  const actions = await page.observe("find submit buttons");

  // Agent - Automate entire workflows
  const agent = stagehand.agent({
      provider: "anthropic",
      model: "claude-sonnet-4-20250514",
      options: {
        apiKey: process.env.ANTHROPIC_API_KEY,
      },
  })
  await agent.execute("apply for this job");
  ```

  ```python Python
  # Act - Execute natural language actions
  await page.act("click the login button")

  # Extract - Pull structured data
  result = await page.extract(
    schema={"price": float}
  )

  # Observe - Discover available actions
  actions = await page.observe("find submit buttons")

  # Agent - Automate entire workflows
  await agent.execute("apply for this job")
  ```
</CodeGroup>

## 开发者为何选择 Stagehand

- **精确控制**：将 AI 驱动的动作与确定性代码相结合。你可以精确决定使用多少 AI。

- **真正可复现**：动作可被精确保存与重放。浏览器自动化不再出现“只在我机器上能跑”的问题。

- **可规模化维护**：一个脚本即可自动化多个网站。站点变更时，你的自动化会随之适配。

- **可组合工具**：通过 act()、extract()、observe() 和 agent() 自由选择自动化程度。
