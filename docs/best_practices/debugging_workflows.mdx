
### Debugging Workflows

**Step 1: Enable Debug Logging**
```typescript
const stagehand = new Stagehand({
  env: "LOCAL",
  verbose: 2,
  localBrowserLaunchOptions: { headless: false, devtools: true }
});
```

**Step 2: Analyze Action Failures**
- Check if elements exist with `observe()` before acting
- Verify selectors are correct and unique
- Ensure page has fully loaded before actions

**Step 3: Monitor Performance**
- Track execution times for optimization
- Identify slow operations and bottlenecks
- Compare local vs Browserbase performance

**Step 4: Review LLM Interactions**
- Check token usage and costs
- Analyze model selection effectiveness
- Monitor cache hit rates
